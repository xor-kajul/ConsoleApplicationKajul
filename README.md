Number Guessing Game
<hr>
#Author - [Kajul Verma]
<br>
This is simple javascript-based number guessing program where user has to guess the number. If the number is guessed correctly , the output will be 'you guessed correctly' else  it will keep asking the number. 

<br>
++++++++++++++++++++++++++++++++++++++++++++++
How It Works
<br>
1.The user is prompted with the popup of browser.
2.Guessed number needs to be entered in the popup.
3. If Guess is in correct , the console will return - "Incorrect Guess, Try Again".
4. If Guess in correct, the console will return - "Guess Correct".
5. The program will continue until the correct guess is entered by user. 

<br>
++++++++++++++++++++++++++++++++++++++++++++++++
How to Run
1.Clone this repository.
2.Navigate to the directory where the program file is located.
3.Open the index.html in any browser with console open in browser
